A collection of custom Gutenberg blocks for Affiliate Products Review.

== Description ==

**Affiliaterg Gutenberg Blocks** is a collection of custom Gutenberg Blocks developed with **Gutenberg Native Components**. You can easily showcase your **Affiliate Products** using these blocks and boost your sale. For Example: Amazon Affilaite Products. It comes with  different styles and settings so that you can easily customize it as per your needs.

### Overview

### Key Features
* **Show affiliate product easily like Amazon Products etc**.
* Simple, light-weight, and super-fast
* **Built with only Gutenberg Native Components**
* **No Block Builder at all**
* **100% Responsive**
* **Eye-catching different style layouts**
* **Product List View**
* **Eye-catching Props and Cons**
* **Rating System**
* Custom Color Option
* **Easy to Use in Gutenberg Editor**
* **Drag & Drop Use**

### Blocks List

* [Button](https://wpcenter.io): It is an advanced button with a lot of customization options.
* [Props Cons](https://wpcenter.io): It shows a product's props and cons.
* [Revew Style 1](https://wpcenter.io): Single Product Review style. Tt shows product's information in a separate view.
* [Revew Style 2](https://wpcenter.io): Single Product Review style. It shows product's information in a single box.
* [Revew List](https://wpcenter.io): Single Product Review List style. It shows product information in a list view.
* [List Table](https://wpcenter.io): Collection of Products. It shows multiple products information in a list table view.
* [Comparison Table](https://wpcenter.io): Shows a Products Comparison Table. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/affiliate-products-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Simply go to Gutenberg Block Editor and you will find a new category with a new Block named "Affiliate Products Showcase Block"
4. Drag and drop to use it.
